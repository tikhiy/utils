(static) extend(__super__, __proto__) → {function}

Создает дочерний класс наследованный от `__super__` и расширенным `__proto__`.
Parameters:
Name Type Description
__super__ function Дочерний класс (функция). Не может быть ES2015 классом, стрелочной функцией, или генератором! Только обычная функция: `function __super__ () {}`.
__proto__ object Объект который содержит конструктор дочернего класса и свойства/методы которые надо добавить в прототип нового класса.
